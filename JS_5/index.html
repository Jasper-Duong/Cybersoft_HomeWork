Bài 2

- Input: Số Kw điện (nKW)
- Output: Tiền điện tương ứng (bill)
- Thuật toán:
  if (50 >= nKW): bill = nKW * 500;
  if (100 >= nKW > 50): bill = 50 * 500 + (nKW - 50) * 650;
  if (200 >= nKW > 100): bill = 50 * 500 + 50 * 650 + (nKW - 100) * 850;
  if (350 >= nKW > 200): bill = 50 * 500 + 50 * 650 + 100 * 850 + (nKW - 200) * 1100;
  else: bill = 50 * 500 + 50 * 650 + 100 * 850 + 200 * 1100 + (nKW - 350) * 1300;

Bài 4

- Input: 3 cạnh tam giác (AB, AC, BC)
- Output: Loại tam giác (vuông/cân/đều/khác)
- Thuật toán:
  + Nếu AB = AC = BC => Tam giác đều
  + Nếu (AB = AC) hoặc (AC = BC) hoặc (AB = BC) => Tam giác cân
  + Nếu AB^2 = AC^2 + BC^2 hoặc
    AC^2 = AB^2 + BC^2 hoặc
    BC^2 = AB^2 + AC^2
    => Tam giác vuông
  + Nếu ko thuộc các trường hợp trên => Tam giác là tác giam!!!.
