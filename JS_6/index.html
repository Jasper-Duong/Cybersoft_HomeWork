Bài 1

- Input: Ko có input
- Output: Số nguyên dương (n) nhỏ nhất thoả đk: 1 + 2 + ... + n > 10000
- Thuật toán: + Cho n = 0; + Gọi sum = 0 là tổng của các số 1, 2,..., n.
+ Khi 10000 >= sum:
  + n = n + 1.
  + sum = sum + n;
+ else: Xuất n và dừng chương trình.
