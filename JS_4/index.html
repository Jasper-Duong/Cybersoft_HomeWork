Bài 4

- Input: 3 cạnh tam giác (AB, AC, BC)
- Output: Loại tam giác (vuông/cân/đều/khác)
- Thuật toán:
  + Nếu AB = AC = BC => Tam giác đều
  + Nếu (AB = AC) hoặc (AC = BC) hoặc (AB = BC) => Tam giác cân
  + Nếu AB^2 = AC^2 + BC^2 hoặc
    AC^2 = AB^2 + BC^2 hoặc
    BC^2 = AB^2 + AC^2
    => Tam giác vuông
  + Nếu ko thuộc các trường hợp trên => Tam giác là tác giam!!!.
